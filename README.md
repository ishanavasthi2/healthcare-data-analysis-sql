# 🏥 Healthcare Data Analysis Using SQL (MySQL)

## 📌 Project Overview

This project demonstrates SQL skills by analyzing healthcare data using MySQL.
The project includes database design, table relationships, analytical queries, and healthcare insights generation.

---

## 🚀 Skills Used

* SQL (MySQL)
* Joins
* Aggregate Functions
* CASE Statements
* GROUP BY
* Data Analysis
* Database Design

---

## 📂 Database Tables

* Patients
* Diagnoses
* Outcomes
* Labs

---

## 📊 Key Analysis Performed

### 1. Detailed Patient Lab History

Analyzed patient diagnoses, outcomes, and lab records.

### 2. Average Lab Results by Diagnosis

Calculated average lab test results for each diagnosis type.

### 3. Abnormal Lab Result Detection

Identified patients with abnormal Blood Sugar, Cholesterol, and Hemoglobin levels.

### 4. Treatment Cost Analysis

Calculated total treatment cost by diagnosis.

### 5. High-Risk Patient Identification

Detected high-risk patients based on age, gender, and treatment outcome.

### 6. Lab Trends Over Time

Tracked lab test trends using admission dates.

### 7. Outcome Distribution Analysis

Compared recovery outcomes across diagnoses.

### 8. Lab Result Classification

Classified lab results into High, Low, and Normal categories using CASE statements.

## 📈 Project Outcome

This project demonstrates practical SQL querying, healthcare data analysis, and database management skills suitable for Data Analyst and SQL Developer roles.
